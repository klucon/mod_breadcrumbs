# mod_breadcrumbs

Drobečková navigace odvozená z publikovaných položek `com_menu`.

## Účel

Modul vykreslí breadcrumb podle aktuální URL a stromu položek menu. Je vhodný do pozice `breadcrumbs` v šabloně.

## Poznámky k verzi

První verze s výběrem menu podle aliasu nebo ID a HTML výstupem Bootstrap breadcrumb.
